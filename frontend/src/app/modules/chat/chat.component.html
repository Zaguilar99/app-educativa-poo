Ejercicio: Sistema de Gestión de Vehículos

Crea un sistema de gestión de vehículos utilizando los principios de Programación Orientada a Objetos (POO) en Java. El objetivo es implementar una jerarquía de clases que represente diferentes tipos de vehículos y demostrar el uso de herencia, encapsulación y polimorfismo.

Objetivo: Definir una clase base que contenga atributos y métodos comunes a todos los vehículos.
